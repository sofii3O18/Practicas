Sección 1: Historia del HTML

HTML fue creado por Tim Berners-Lee en 1991 como un lenguaje para compartir documentos científicos en internet.

Desde sus inicios como un lenguaje muy simple, ha crecido hasta convertirse en el pilar fundamental de la web moderna.

Sección 2: Versiones de HTML

HTML ha evolucionado desde HTML 1.0 hasta HTML5, añadiendo en cada versión nuevas capacidades para multimedia, formularios y APIs.

HTML5, lanzado oficialmente en 2014, introdujo etiquetas semánticas y soporte nativo para video y audio.

Sección 3: El Futuro del Web

Las tecnologías web continúan evolucionando con WebAssembly, Progressive Web Apps y Web Components.

La inteligencia artificial está transformando la manera en que los usuarios interactúan con las páginas web.
